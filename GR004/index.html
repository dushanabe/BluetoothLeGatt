Gauges Reality GR004

ScComponents for Android
Gauges are a powerful way to display velocity, status, or simple measurements. This library use the ScGauges library and allow to customize the gauges in every parts. The gauges is touch response and you can use it also as seek-bar. You can use every kind of values, integer or decimal and define the label/values format.

Some main cusomization than can be reached easily from the GR004 class are:
  • Texts
  • Font, size and label/values format
  • Ticks: major and minor
  • Values: current, min and max
  • ...
Download now the demo from Google play store and try all its potential.

Import the library in Android Studio

1) Locate the .AAR file inside the root/library folder.
2) Right click on your project and select "Open Module Settings".

3) Click the ADD (+) button in the top left corner of window to add a new module.

4) Select "Import .JAR or .AAR Package" and click the "Next" button.

5) Keep the app's module selected and click on the Dependencies pane to add the new module as a dependency.
NOTE: in some case Android Studio fail to update the "settings.gradle" file. If happen you'll not see the libraries listed when you press on the "+" button. To fix this issue is enough to open the "settings.gradle" file and manually add the libraries references. For example, if you are importing the libraries GR004 and GR008 in your app, inside your "settings.gradle" file you should have something like: include ':app', ':GR004', ':GR008'.

6) If you have some problems or just want see a well formatted example app with imported libraries please download our "example" from the link: SIMPLE APPLICATION EXAMPLE

Import the library in Eclipse

It is not possible to import directly an aar file into eclispe. The AAR is handled specifically by Android Studio through the Android gradle plugin.
However, there are several way to import them manually inside eclipse.

1) Locate the .AAR file inside the root/library folder.
2) Rename the .aar in .zip and extract it.
3) Inside the folder, you will find the resources and a class.jar.
4) Copy these files inside Eclipse and done.
